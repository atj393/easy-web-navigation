# Store assets

Generated, original brand artwork and placeholders for store listings. No
external or copyrighted assets are used.

## Real artwork (ready to use)

| File                             | Dimensions | Use                               |
| -------------------------------- | ---------- | --------------------------------- |
| `chrome-small-promo-440x280.png` | 440×280    | Chrome Web Store small promo tile |
| `edge-logo-300x300.png`          | 300×300    | Microsoft Edge Add-ons logo       |

These are generated by `scripts/generate-store-assets.mjs` (pure Node, no
dependencies) by downscaling the canonical brand source
`assets/brand/easy-web-navigation-icon-source.png`. The promo tile is honest
brand artwork (the real icon + the product name) — not a screenshot. Re-run with
`node scripts/generate-store-assets.mjs`. The toolbar icons (16/32/48/128) live
in `apps/extension/public/` and are downscaled from the same source by
`apps/extension/scripts/generate-icons.mjs`.

## Placeholders (MUST be replaced before submission)

| File                                  | Note                                                    |
| ------------------------------------- | ------------------------------------------------------- |
| `screenshot-placeholder-1280x800.svg` | Clearly-labeled placeholder. **Not** a real screenshot. |

**Product screenshots are not committed and were not auto-generated** — they must
be captured manually from a real browser session so they show genuine behavior.
Follow [`../screenshot-plan.md`](../screenshot-plan.md) for the exact shot list,
then export real PNGs (1280×800) into this folder and remove the placeholder
before submitting to any store.

Do not submit placeholder images to a store.
