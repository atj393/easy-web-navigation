# Privacy Policy

The canonical, store-ready privacy policy for Easy Web Navigation lives at
[`docs/store/privacy-policy.md`](store/privacy-policy.md).

Summary: Easy Web Navigation processes pages **locally** to report keyboard accessibility findings.
It collects **no** data — no analytics, no tracking, no AI calls, no remote API calls, no account.
Only your preferences are stored locally. No page content is uploaded. See the full policy linked
above.
